# Abridgment Model

## Thesis

The abridgment system should preserve choice, not enforce a single editor's cut. A chapter can be included fully, excerpted, summarized, deferred, or hidden for a selected path. The reader should always be able to recover the skipped material.

## Inclusion Levels

| Level | Meaning |
|---|---|
| `required` | Needed for the path to make sense. |
| `recommended` | Strongly useful, but not strictly required. |
| `optional` | Good enrichment for students with more time. |
| `summary-only` | Skipped in full prose, represented by a summary. |
| `defer` | Hidden initially, easy to expand. |

## Starter Paths

### Narrative Core

The shortest coherent route through the plot. It should include the shore opening, Ishmael and Queequeg's bond, the signing of the Pequod, Ahab's declared quest, major warnings or escalations, and the final chase.

### Classroom Standard

The likely default for a teacher. It should preserve enough of Melville's weirdness to make the book itself visible: Ahab, Starbuck, Queequeg, the crew, the sermon, selected cetology, gams, labor chapters, symbolic chapters, and the ending.

### Thematic Trails

Trails can be read across the full book:

- Ahab and Starbuck.
- Ishmael and Queequeg.
- Whaling labor.
- Cetology and classification.
- Gams and foreshadowing.
- Biblical and classical allusion.
- Race, empire, and global crew.
- Theater, sermons, soliloquies, and formal experiments.
- Law, property, and political metaphor.

## Metadata Judgment

Gemini's preliminary plan is useful as a starting inventory, not a governing authority. Chapter classification should be checked against the actual text and revised as the project matures.
