# Schemas

JSON Schemas for the project content contracts.

These are starter contracts. Tighten them as the data model becomes real.

Current scale-up contracts include annotation display metadata, evidence, provenance, review queues, relationship targets, the annotation layer registry, the trail/entity taxonomy, and the geography registry. Custom validation in `scripts/validate/validate-basic.mjs` is still the main enforcement layer for generated indexes and cross-file references.
